# Умный указатель
На основе подсчета ссылок

Ссылка ассоциирована с объектом

Методы:
 - ```Get``` - Получить объект
 - ```Set``` - Установить объект
 - ```Release``` - Освободить указатель от владения объектом
 
Также реализовать перегрузку операций:
 - ```=```
 - ```->```
 - ```*```
